## Quick Reference

### Tech Stack Summary

- **Router:** chi v5.2.3
- **Database:** PostgreSQL + pgx v5 + sqlc
- **Cache:** Redis v9
- **Job Queue:** Asynq v0.25.1
- **Tracing:** OpenTelemetry v1.39
- **Metrics:** Prometheus
- **Logging:** Zap

### Entry Points

| Entry Point | Purpose | Command |
|-------------|---------|---------|
| `cmd/server` | HTTP API | `go run ./cmd/server` |
| `cmd/worker` | Background jobs | `go run ./cmd/worker` |
| `cmd/scheduler` | Cron jobs | `go run ./cmd/scheduler` |
| `cmd/bplat` | CLI scaffolding | `./bin/bplat` |

## Getting Started

### For Developers

1. **Setup environment:**
   ```bash
   cp .env.example .env
   docker-compose up -d
   ```

2. **Run migrations:**
   ```bash
   make migrate-up
   ```

3. **Start development server:**
   ```bash
   make dev
   ```

4. **Explore GraphQL:**
   Open `http://localhost:8080/playground`

### For AI Assistants
